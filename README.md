specfiles

Check these with rpmlint.

- http://docs.fedoraproject.org/en-US/Fedora_Draft_Documentation/0.1/html/RPM_Guide/ch-creating-rpms.html
- http://www.ibm.com/developerworks/library/l-rpm1/
